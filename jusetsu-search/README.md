# 住設サーチ（Jusetsu Search）

住宅設備機器に特化した検索・比較サイトの**設計ドキュメント**一式です。
[建材サーチ](https://kenzai-search.jp) を参考に、住宅設備の情報を一箇所に集約し、
**メーカーサイトを何社も見なくても住設サーチだけで比較検討できる**ことを目指します。

## ターゲットユーザー

| ユーザー | 主な利用シーン |
|---|---|
| 住宅営業 | 商談中に施主へ商品提案・比較資料を即提示 |
| 住宅設計 | 仕様検討・CADデータ取得・図面落とし込み |
| 工務店 | 採用設備の選定・カタログ/仕様書の一括取得 |
| リフォーム会社 | 既存設備からの置換候補比較 |
| 施主 | 入れたい設備のイメージ収集・施工事例閲覧 |

## サービスコンセプト

> 住宅設備の情報を一箇所に集約する検索サイト。
> メーカーサイトを横断せず、住設サーチだけで比較検討を完結させる。

## 対象カテゴリ

キッチン / ユニットバス / 洗面化粧台 / トイレ / 給湯設備 / 換気設備 / 空調設備 /
太陽光発電 / 蓄電池 / 宅配ボックス / 照明

## 想定技術スタック（提案）

| レイヤ | 技術 |
|---|---|
| フロントエンド | Next.js (App Router) + TypeScript + Tailwind CSS |
| 状態管理 | React Query（サーバ状態）/ Zustand（お気に入り・比較） |
| バックエンド | Node.js + NestJS（or Express）/ REST API |
| データベース | PostgreSQL（全文検索は pg_trgm or Meilisearch） |
| ファイル保管 | S3 互換ストレージ（カタログ/CAD/画像） |
| 認証 | 管理者：メール+パスワード(JWT) / 一般：匿名+任意登録 |
| インフラ | Vercel（FE）+ Render/Fly.io/AWS（API・DB）+ CDN |

## デザイン原則

- シンプル・見やすい・検索しやすい
- スマホ完全対応のレスポンシブ
- 余白を活かしたレイアウト
- 住宅会社の営業担当が毎日使いたくなるUI（最短2クリックで比較表へ）
