# 3. 画面一覧

建材サーチに揃え、(1) 登録不要で全機能、(2) 施工例の逆引きを主役、(3) メーカー入稿＋運営承認、
(4) 検索サジェスト を反映した全画面の定義。

## 3.1 一般ユーザー向け画面（ログイン不要）

| ID | 画面名 | パス | 主な要素 | 主な遷移先 |
|---|---|---|---|---|
| P-01 | トップページ | `/` | 検索バー(サジェスト) / 人気カテゴリ / 人気メーカー / 人気商品 / 新着商品 / **施工例** | 商品一覧, 商品詳細, メーカー詳細, 施工例 |

## 3.2 マイページ（任意登録・同期用）

| ID | 画面名 | パス | 主な要素 |
|---|---|---|---|
| M-00 | ログイン / 新規登録 | `/mypage/login`, `/mypage/register` | メール / パスワード（任意。未登録でも全機能は使える旨を明記） |
| M-01 | お気に入り一覧 | `/mypage/favorites` | お気に入り商品グリッド / 一括比較 / 削除（匿名分はログイン時に自動同期） |
| M-02 | 比較リスト | `/mypage/compare` | 保存した比較セット一覧 → 比較ページを再現 |

> 未ログインでも ♡お気に入り・比較は `anon_id` で動作。ログインすると端末間で同期される。

## 3.5 画面共通要素

- **ヘッダー**：ロゴ、常設検索バー（**サジェスト**：メーカー/商品/シリーズ）、グローバルナビ（商品/メーカー/**施工例**）、比較バッジ、お気に入りバッジ
- **比較トレイ**：画面下部に固定。比較追加した商品サムネ表示＋「比較する」ボタン（最大4、超過時は警告トースト）
- **フッター**：カテゴリ／メーカー／施工例／規約等／「メーカー様へ（入稿案内）」
- **トースト**：お気に入り追加、比較追加/上限超過、入稿の申請・承認・差戻し
- **空状態**：検索0件、お気に入り0件、施工例0件のガイド表示
- **価格表示の出し分け**：価格あり→金額（税抜）、なし→「オープン価格」「要問合せ」バッジ
