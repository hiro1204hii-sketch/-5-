# 3. 画面一覧

## 3.1 一般ユーザー向け画面

| ID | 画面名 | パス | 主な要素 | 主な遷移先 |
|---|---|---|---|---|
| P-01 | トップページ | `/` | 検索バー / 人気カテゴリ / 人気メーカー / 人気商品 / 新着商品 / 施工事例 | 商品一覧, 商品詳細, メーカー詳細, 施工事例 |
| P-02 | 商品一覧 | `/products` | 検索バー / 絞り込みサイドバー（カテゴリ・メーカー・価格帯・タグ）/ 並び替え / 商品カードグリッド / ページネーション / 比較・♡ボタン | 商品詳細, 比較 |
| P-03 | カテゴリ別一覧 | `/categories/:slug` | カテゴリ説明 / そのカテゴリ専用タグ絞り込み / 商品グリッド | 商品詳細 |
| P-04 | 商品詳細 | `/products/:id` | 画像ギャラリー / メーカー・シリーズ・商品名 / 価格・サイズ・カラー / 商品説明 / 仕様表 / 資料DL（カタログ・仕様書・CAD・取説）/ 商品URL / 施工事例 / 比較・♡ / 関連商品 | メーカー詳細, 比較, 施工事例詳細 |
| P-05 | メーカー一覧 | `/makers` | 検索 / 50音・カテゴリフィルタ / メーカーカード（ロゴ・商品数） | メーカー詳細 |
| P-06 | メーカー詳細 | `/makers/:id` | メーカー概要 / ロゴ / 公式サイトリンク / 主要商品 / 全商品一覧（絞り込み付） | 商品詳細 |
| P-07 | 比較ページ | `/compare` | 最大4商品の比較表（メーカー/シリーズ/価格/サイズ/カラー/主な特徴/仕様）/ 商品追加・削除 / 列の入替 | 商品詳細 |
| P-08 | 施工事例一覧 | `/cases` | 絞り込み（カテゴリ・メーカー・タグ）/ 事例カードグリッド | 施工事例詳細 |
| P-09 | 施工事例詳細 | `/cases/:id` | 写真ギャラリー / 商品名 / カラー / コメント / 使用商品へのリンク | 商品詳細 |
| P-10 | お気に入り | `/favorites` | お気に入り商品グリッド / 一括比較ボタン / 削除 | 商品詳細, 比較 |
| P-11 | 検索結果 | `/search` | 検索キーワード / メーカー・商品・シリーズの横断結果（タブ or セクション） | 商品詳細, メーカー詳細 |
| P-12 | サービスについて等 | `/about` 他 | 静的コンテンツ | - |

## 3.2 管理画面

| ID | 画面名 | パス | 主な要素 |
|---|---|---|---|
| A-00 | ログイン | `/admin/login` | メール / パスワード |
| A-01 | ダッシュボード | `/admin` | 登録件数サマリ（商品/メーカー/事例）/ 最近の更新 / 公開待ち |
| A-02 | 商品一覧 | `/admin/products` | 検索・絞り込み / 一覧テーブル / 公開状態 / 編集・削除 |
| A-03 | 商品登録 | `/admin/products/new` | メーカー・カテゴリ選択 / 商品名・シリーズ・説明 / 価格・サイズ・カラー / 画像アップロード / 資料アップロード（種別選択）/ 仕様（キー・バリュー追加）/ タグ選択 / 商品URL / 公開設定 |
| A-04 | 商品編集 | `/admin/products/:id/edit` | A-03 と同等（既存値プリフィル） |
| A-05 | メーカー一覧 | `/admin/makers` | 一覧 / 編集・削除 |
| A-06 | メーカー登録/編集 | `/admin/makers/new`, `/:id/edit` | 名称 / 概要 / ロゴ / 公式URL |
| A-07 | カテゴリ・タグ管理 | `/admin/taxonomies` | カテゴリCRUD / タグCRUD / カテゴリ⇔タグ紐付け |
| A-08 | 資料管理 | `/admin/assets` | 商品横断の資料一覧 / 差し替え / 種別管理 |
| A-09 | 施工事例一覧 | `/admin/cases` | 一覧 / 編集・削除 |
| A-10 | 施工事例登録/編集 | `/admin/cases/new`, `/:id/edit` | 紐付け商品 / 写真 / カラー / コメント / 公開設定 |

## 3.3 画面共通要素

- **ヘッダー**：ロゴ、常設検索バー、グローバルナビ、比較バッジ、お気に入りバッジ
- **比較トレイ**：画面下部に固定。比較追加した商品サムネを表示し「比較する」ボタン
- **フッター**：カテゴリ／メーカー／規約等リンク
- **トースト**：お気に入り追加、比較追加（上限4件超過時の警告）
- **空状態**：検索ヒット0件、お気に入り0件のガイド表示
