# 4. UIワイヤーフレーム

ASCIIによる主要画面のレイアウト案。PC（デスクトップ）を基本とし、スマホ（SP）は
1カラム化・絞り込みはドロワー化する。余白を活かし、検索を常時最上部に固定。
