# 住宅概算システム（スタンドアロン版）

住宅会社向けの **高精度な概算見積システム**。
工事項目を「業者単価 × 数量 × 掛け率」で積み上げ、正式見積に近い概算を素早く出すことを目的とする。

- 形態: **HTML 1ファイル**（`standalone/住宅概算システム.html`）。サーバー不要・ダブルクリックで起動。
- 保存: ブラウザの localStorage ＋ パソコンのファイルへ自動保存（File System Access API）。
- オンライン前提（Excel/PDF取込時に解析ライブラリをCDNから取得）。

> フルスタック版（React + Node + SQLite）は方針転換により**廃止**しました。開発・判断はこのスタンドアロン版に一本化します。

## 使い方
1. `standalone/住宅概算システム.html` をダウンロードしてダブルクリック。
2. 初回は空。⚙ バックアップ/設定 →「📦 サンプルデータを読み込む」で例を確認できます。
3. ⚙ →「💾 保存先ファイルを設定」で、以降の入力をPCのファイルへ自動保存。

## 本質的な目的
- お客様への初回概算と、業者見積を集めた正式見積の **金額差を減らす**。
- 概算精度が上がることで「建物を小さく/中止」を防ぎ、前向きな提案（大きく・良い仕様）につなげる。

## コア概念
- **工事項目マスター**：大項目 → 中項目 → 小項目①②③ の5階層。
- **掛ける相手**：延床/建築/敷地（㎡・坪）・一式・手入力・独自項目（外壁面積/各種「〜代」等）。
- **計算**：原価 = 業者単価 × 数量、売価 = 業者単価 × 数量 × 掛け率、粗利 = 売価 − 原価。
- **単価取込**：業者見積（PDF/Excel/CSV/貼付）から工種・単価を工事項目マスターへ登録（最優先機能）。
- **住設・オプション**：標準仕様との **差額管理**（グレード管理ではない）。
